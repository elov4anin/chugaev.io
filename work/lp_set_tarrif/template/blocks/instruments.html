• 01

    Автоплатеж

    Возможность для плательщика
    настроить автоплатеж по календарю
    или по задолженности.

  • 02

    Инвойсинг

    Выставление счетов плательщикам на электронную почту или в телефон с возможностью моментальной оплаты.

  • 03

    Привязка карты

    Сохранение реквизитов карты
    для быстрой оплаты без повторного ввода ее данных.

  • 04

    Чек

    Чек с полным реквизитным составом
    при любом способе оплаты, кассовый чек

  • 05

    Информационный сервис

    Отображение полной информации о платеже в личном кабинете плательщика.

  • 06

    Защита данных
    банковской карты

    Соблюдение стандарта безопасности данных PCI DSS 3.2. Level 1
