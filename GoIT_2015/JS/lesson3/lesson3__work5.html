Задание 5 (Шахматная доска) Напишите программу, создающую строку, содержащую решётку 8х8, в которой линии разделяются символами новой строки. На каждой позиции либо пробел, либо \#. В результате должна получиться шахматная доска. # # # # # # # # # # # # # # # # # # # # # # # # # # # # # # # # Когда справитесь, сделайте размер доски переменным, чтобы можно было создавать доски любого размера.
